# Portafolio Web – Ingrid Z. Mendoza Dórame

Este proyecto corresponde a mi portafolio personal, desarrollado con HTML, CSS y JavaScript. Su propósito es presentar mi perfil académico y profesional, experiencia, proyectos y medios de contacto de forma clara y responsiva.

## Estructura del proyecto

```
├── index.html
├── assets/
│   ├── css/style.css
│   └── imgs/..
└── js/script.js
```

## Contenido

El sitio incluye las siguientes secciones:
- Header con información personal
- Sobre mí
- Experiencia académica y profesional (acordeón interactivo)
- Fun facts
- Contacto
- Footer

## Tecnologías utilizadas

- HTML5  
- CSS3  
- JavaScript  
- Font Awesome  

## Visualización

Abrir el archivo `index.html` en cualquier navegador web moderno. No requiere instalación adicional.
